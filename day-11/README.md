# Day 11 - Node.js

## Topics
- Node.js architecture and event loop
- Built-in modules (fs, path, os, url, crypto, http)
- Creating an HTTP server
- Custom modules (require/exports)
- File system streams and pipes
- npm packages (lodash, nodemon)

## Folders
- `custom-modules/` - require, exports, module.exports
- `inbuilt-modules/` - fs, path, os, url, crypto, http
- `file-system-stream/` - Streams, pipes, readline
- `npm-modules/` - npm package usage
- `server/` - HTTP server with routing
