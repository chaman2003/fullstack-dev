Chrome DevTools

Console Tab

See logs, errors, and run JS code live.

console.log("Normal message");
console.warn("Warning message");

console.table([{name:"John", age:30}, {name:"Jane", age:25}]);

Elements Tab

Inspect and edit HTML/CSS in real time.

  • Right-click → Inspect Element
  • Edit HTML directly in the panel
  • Change CSS values and see updates live

Network Tab

See all network requests, status codes, load times.

  • Check if API calls are working (200 vs 404 vs 500)
  • See request payload and response data
  • Check load times for images and scripts

Sources Tab

Set breakpoints and debug JS step by step.

Using Breakpoints

function calculateTotal(prices) {
    let total = 0;
    for (let i = 0; i < prices.length; i++) {
        total += prices[i];  // Put breakpoint here
    }
    return total;
}

const result = calculateTotal([10, 20, 30, 40]);
console.log(result);

How: Open Sources tab → Find your JS file → Click line number to add breakpoint → Reload page

Common JS Errors

ReferenceError: x is not defined 
console.log(x); // x was never declared
Fix: Declare the variable first: let x = 5;
TypeError: Cannot read property of undefined 
const user = null;
console.log(user.name); // user is null
Fix: Check if it exists first: if (user) console.log(user.name);
NaN (Not a Number) 
let result = "hello" - 5; // NaN
Fix: Make sure you're doing math on numbers

CSS Debugging Tips

  • Add border: * { border: 1px solid red; } to see all element boxes
  • Check specificity: A more specific selector overrides a less specific one
  • Inheritance: Not all CSS properties are inherited (e.g. width, height, margin aren't)
  • DevTools: Check "Computed" tab to see final CSS values
  • :hover not working? In Elements tab, right-click element → Force State → :hover
