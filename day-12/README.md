# Day 12 - Express.js

## Topics
- Express app setup
- Routing (GET, POST, PUT, DELETE)
- Route parameters and query strings
- Middleware (app-level, error handling)
- Form handling with multer
- Static file serving
- Controllers and routers (modular)

## Folders
- `basic/` - Hello world Express app
- `routes/` - Basic routing
- `controller/` - Modular routes with controllers
- `http-methods/` - CRUD operations
- `middleware/` - App-level and error middleware
- `handling-form-data/` - URL-encoded and multipart forms
- `server-static-files/` - Serving static assets
